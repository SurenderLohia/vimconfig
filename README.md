## My Vim configuration files


### How to use my files

```bash
$ cd
$ git clone git://github.com/luckydev/vimconfig.git ~/vimconfig
$ ln -s ~/vimconfig ~/.vim
$ ln -s ~/vimconfig/vimrc ~/.vimrc
$ ln -s ~/vimconfig/gvimrc ~/.gvimrc
$ cd ~/vimconfig
$ git submodule init
$ git submodule update

```

Done. You are all set now.
